PREREQUISITES
=============

cmptcomplexity requires the following software installed for your platform:
0) UNIX-like system supporting signal communication

1) Python__ 2.7 or >= 3.4

__ http://www.python.org

2) NumPy__ >= 1.8.2

__ http://www.numpy.org/

3) MatPlotLib__

-- http://matplotlib.org/
